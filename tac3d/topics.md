# TAC-3D Content Strategy

## How to generate a screening
1. Search current news for a real biotech, investment, or strategic decision happening this week
2. Use the real decision as the TAC-3D input
3. Build 2-3 evidence layers from actual publicly available data
4. Call TAC-3D API from TOOLS.md
5. Use the real verdict and scores as the basis for all posts

## Post frequency and format

### LinkedIn (weekly)
Professional narrative. 150-300 words. Lead with the structural verdict. Include scores.

### X (daily)
Max 300 chars. Hook + verdict + one insight. No hashtag spam.

### Bluesky (daily)
Max 300 chars. More technical tone than X.

### Substack (weekly)
800-1200 words. Deep analysis. Use TAC-QSW framework. Link to whitepaper.

### Reddit (bi-weekly)
200-400 words. Conversational. Question-led. Target r/investing or r/biotech.
